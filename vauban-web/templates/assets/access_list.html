About Access Rules

Access rules define the permissions for users and groups to connect to assets. Each rule can specify:

  • Which users or groups can access which assets
  • Time-based restrictions (allowed hours/days)
  • Whether approval is required before connection
  • Session recording requirements
  • Maximum session duration
